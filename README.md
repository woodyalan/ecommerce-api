# Ecommerce API

Aplicação desevolvida durante a disciplina de Node.js do Bootcamp Viptech 2022.

## Antes de iniciar a aplicação

Instale as dependências do projeto:

`yarn install`

ou

`npm install`

## Iniciar a aplicação

Para inciar a aplicação, execute em seu terminal: 

`yarn start`

ou

`npm start`
